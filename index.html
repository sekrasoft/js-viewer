Краткая справка

Управление с клавиатуры:
  • Left/Right - предыдущее/следующее изображение
  • Up/Down - повернуть изображение на 90 градусов против/по часовой стрелке
  • +/- - увеличить/уменьшить изображение
  • 0 - размер по умолчанию
  • Home/End - первое/последнее изображение
  • s - запустить/остановить показ слайдов
  • f - развернуть на полный экран/свернуть обратно
  • Alt+Enter - развернуть на полный экран/свернуть обратно
Управление мышью:
  • Клик по левой/правой области - предыдущее/следующее изображение
  • Навести мышь вниз - открыть панель с кнопками:
    • ► - запустить/остановить показ слайдов
    • ⇦/⇨ - предыдущее/следующее изображение
    • ↶/ ↷ - повернуть изображение на 90 градусов против/по часовой стрелке
    • +/- - увеличить/уменьшить изображение
    • ◼ - размер и положение по умолчанию
    • ⇗ - развернуть на полный экран/свернуть обратно
  • Навести на кнопку - получить подсказку
  • Навести мышь вверх - открыть справку
  • Перетаскивание изображения - перетаскивание изображения :)
  • Shift+wheel - изменение масштаба изображения
Управление тачскрином:
  • [план] Перетаскивание изображения - перетаскивание изображения :)
  • [план] Шорк влево/Шорк вправо - предыдущее/следующее изображение
  • [план] Два пальца - изменение масштаба изображения
Список фотографий на странице обновляется автоматически.
Внимание! Следует использовать браузер Google Chrome для адекватного отображения.
